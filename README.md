## 如何使用

`git clone https://github.com/jingxiang0405/VisFinal`

`cd VisFinal`

`npm i`

`npm run dev`

共有三個操作區(如下圖)：
區塊(1)—類別操作區：選擇想檢視的統計數據。如：交岔路、單路部分
區塊(2)—地圖操作區：可以點選縣市以檢視該縣市更詳細的數據

針對每次操作，都會於區塊(3)顯示對應的圖表
